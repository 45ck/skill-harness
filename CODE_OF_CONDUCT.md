# Code Of Conduct

This project expects focused, respectful collaboration.

## Standards

Expected behavior:

- be specific, factual, and constructive
- keep feedback tied to the work
- respect maintainer scope decisions
- disclose uncertainty and evidence gaps
- avoid sharing private data, credentials, or sensitive customer information

Unacceptable behavior:

- harassment, threats, or discriminatory language
- personal attacks or repeated bad-faith disruption
- publishing private security details before maintainers can respond
- sharing private data or secrets without permission

## Reporting

Report conduct concerns through GitHub project moderation tools or by requesting a private maintainer contact in a GitHub issue. Do not include sensitive personal information in a public issue.

Maintainers may remove comments, close threads, block users, or restrict participation when needed to keep the project usable and safe.
