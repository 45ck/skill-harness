# demo-production-skills

Embedded pack for source-backed demo, QA, and media production workflows.

Included skills:

- `demo-story-packager` - assemble a completed demo run into source, media, evidence, and handoff artifacts
- `demo-social-cut` - plan short silent cuts and slideshow-style edits from existing demo evidence
- `demo-slideshow-edit` - plan no-caption slideshow MP4s and frame reels from screenshots, storyboards, and selected spans
- `demo-review-surface` - create static review surfaces that compare demo media, source specs, and quality evidence
- `qa-to-demo` - convert manual QA reports or findings into reproducible demo specs or clip plans
- `demo-release-packager` - assemble approved media, source, evidence, and promotion notes into a safe handoff bundle

Use this pack for media workflows that start from structured browser capture, QA evidence, or demo-machine output. Do not use it as a general-purpose video editing pack.
