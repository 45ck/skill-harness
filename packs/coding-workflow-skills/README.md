# coding-workflow-skills

Embedded pack for GitHub-native execution and review flow.

Use this pack when work needs issue-driven delivery, CI triage, PR review follow-through, or a final review-ready check.

Install it through the harness:

```bash
./skill-harness install --packs=coding-workflow-skills --packs-only
```

Prerequisites depend on the selected skill. GitHub-focused skills expect repository access and the relevant GitHub CLI, app, or browser context.

Included skills:

- `issue-driven-delivery`
- `gh-review-followthrough`
- `gh-actions-failure-triage`
- `review-ready-check`
